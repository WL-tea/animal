# 知识点：JS 要学到什么程度

## 不需要背关键字

常用的一共就十几个（const、function、if、else、return），用多了自然记住，不是靠背的。

## 需要具备的能力

1. **读代码**：能猜出这段代码大概在干什么
2. **描述问题**：代码出错了能说清楚症状
3. **提需求**：说清楚想要什么效果

具体怎么写——AI 可以帮你写。

## CSS 和 JS 的分工

| 谁 | 管什么 | 我们的例子 |
|----|--------|-----------|
| CSS | **静态长相** | 大小、颜色、形状、眼睛位置、走路晃动动画 |
| JS | **动态行为** | 走到哪、被拖拽、双击弹气泡 |

CSS 决定宠物长什么样，JS 决定宠物做什么。

## 代码阅读技巧

一段 JS 代码的核心逻辑通常只有几行，其他都是"保镖"代码（防出错、处理边界）。

pet.js 的核心就是 3 行：
```js
state.x += state.vx;        // 走一步
if (state.x <= 0) state.vx *= -1;  // 碰墙回头
pet.style.left = state.x + "px";   // 刷新位置
```

其他代码都是为这 3 行服务的（限制范围、处理拖拽等）。

## 变量名会说话

```
state.x, state.y      → 位置坐标
state.vx, state.vy    → 速度（velocity）
state.dragging        → 是否在拖拽
limit.maxX            → 最大边界
```

看名字就能猜用途，猜错了也没关系。

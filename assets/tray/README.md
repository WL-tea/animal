# Windows 托盘图标

`tray-icon-source.png` 是可编辑的高分辨率源图。`tray-icon.ico` 是 Windows
运行时首选资源，包含 16、20、24、32、40、48、64、128 和 256 像素图层，
由系统按显示缩放比例选择。相同尺寸的 PNG 文件用于检查单层效果，并作为
ICO 缺失时的开发环境回退资源。

更新图标时：

1. 保持源图为透明背景的正方形 RGBA 图片，并在四周保留一致留白；
2. 使用高质量缩小算法重新导出全部尺寸，不要放大小尺寸图片；
3. 重新生成多尺寸 `tray-icon.ico`；
4. 在 Windows 浅色、深色任务栏和 100%～200% 缩放比例下检查清晰度、边缘和视觉重心；
5. 托盘图标只服务于通知区域，不与安装包、可执行文件或普通窗口图标强制共用。

资源缺失时，应用会记录错误并使用 Electron 的空图像安全降级，不会因此中止启动。
